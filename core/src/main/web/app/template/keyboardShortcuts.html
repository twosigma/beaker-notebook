Keyboard Shortcuts

On input cells

Cell editing Ctrl-C Copy
Ctrl-X Cut
Ctrl-V Paste
Ctrl-Z Undo
Ctrl-Y Redo
Ctrl-Space Code autocomplete (except in emacs mode, then it sets the mark)
Alt-F11 Toggle fullscreen editor
Esc Exit fullscreen editor
Tab At the beginning of a line, then indent. In the middle or end, then autocomplete.
Shift-Ctrl-F Reindent selected region or current line.
Ctrl-/ Toggle comments on current line or selection
Cell Evaluation: Ctrl-Enter Evaluate
Shift-Enter Evaluate and move focus to next cell.
Shift-Ctrl-Enter Evaluate selection or (if no selection) evaluate current line and move cursor to the next line.
Notebook editing: Ctrl-Shift-A Add a new code cell
Ctrl-Shift-Y Add a new text cell
Ctrl-Shift-U Insert a new code cell before the current one
Ctrl-Shift-> Raise section level
Ctrl-Shift-< Lower section level
Ctrl-alt-1 Insert new section with style 1 after the current one
Ctrl-alt-2 Insert new section with style 2 after the current one
Ctrl-alt-3 Insert new section with style 3 after the current one
Ctrl-alt-4 Insert new section with style 4 after the current one
Ctrl-Shift-E Pop up evaluator selector
Ctrl-S Save notebook
Ctrl-Alt-Up Move cell up
Ctrl-Alt-Down Move cell down
Ctrl-Alt-D Delete cell
Ctrl-Alt-Hide Hide cell
Cell navigation: Alt-Down (or Alt-J) Move focus to next code cell
Alt-Up (or Alt-K) Move focus to previous code cell

On the Start Page

Alt-N Create new empty notebook
Alt-Shift-N Create new default notebook
